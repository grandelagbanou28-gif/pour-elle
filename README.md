💍 Project: Eternity - Ophélia
"Parce que certaines promesses ne s'écrivent pas sur du papier, mais se bâtissent avec du code et du cœur."

Ce projet est une plateforme immersive conçue exclusivement pour Ophélia Imane Kèmi MYEHOUENOU. Il marque la transition entre un passé de non-dits et un futur de bâtisseur. Ce n'est pas un simple site vitrine, c'est une déclaration d'intention : celle d'un homme qui ne voit plus "l'amitié" comme une option, mais le mariage comme un objectif.

🌌 Vision Philosophique
Dans un monde de distractions éphémères, ce site utilise la technologie pour figer le temps. Il repose sur trois piliers :

La Clarté : Affirmer ses intentions sans l'ombre d'un doute.

La Maturité : Reconnaître le passé (le collège) pour mieux honorer le futur.

L'Engagement : Transformer des lignes de code en promesses de vie.

🛠️ Stack Technique
Le projet a été développé pour être fluide, précieux et intemporel :

HTML5 / CSS3 (Custom Variables) : Une palette de couleurs "Gold & Wine" pour évoquer la royauté et la profondeur des sentiments.

JavaScript (Vanilla) :

Système de particules 3D sur Canvas pour symboliser l'immensité du destin.

Curseur personnalisé et traînée lumineuse pour une interactivité douce.

Animation dynamique du bracelet "Grandel" pour rappeler le lien physique.

Animations : Intersection Observer API pour un dévoilement progressif des sentiments au fil du défilement (Scroll Reveal).

🎨 Design & Symbolique
Le Bracelet Animé : Représentation du cadeau reçu, pivot central de la reconnaissance émotionnelle.

Le Système d'Étoiles : Twinkling stars en arrière-plan pour illustrer que même dans l'obscurité, l'ambition brille.

Typographie : Utilisation de Playfair Display (sérieux/mariage) et Dancing Script (intimité/émotion).
